Основные теги и примеры для начинающих

1. Основная структура страницы

<!doctype html>
<html lang="ru">
  <head>
    <meta charset="utf-8">
    <title>Название страницы</title>
  </head>
  <body>
    Контент страницы
  </body>
</html>

2. Заголовки и текст

Пример:

Это H1

Это H2

Это H3

Обычный абзац текста с жирным и курсивом.

Перенос строки делается с тегом <br>.

8. Семантические теги

Используй их для структуры:

  • <header> — шапка страницы
  • <nav> — меню
  • <main> — основной контент
  • <section> — раздел
  • <article> — самостоятельная статья
  • <footer> — подвал
